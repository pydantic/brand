# Pydantic Brand Palette & Visual Identity

Source: Brand Guidelines v1.0 (September 2025), pydantic/brand repo, and pydantic.dev website implementation.

## 1. Logo
[SVG](https://github.com/pydantic/brand/blob/main/pydantic-logo-square-1000px-transparent.svg) |
[PNG](https://github.com/pydantic/brand/blob/main/pydantic-logo-square-1000px-transparent.png)


### Logomark
A fusion of Penrose tiling shapes (darts and kites). Represents creativity, precision, and efficiency. Clean lines, harmonious angles.

### Logotype
Set in **IBM plex sans medium**. Always use the logotype asset rather than typing the name, unless in running text.

### Lockup
Logomark + logotype side by side. Preferred format for most uses.

### Clearance
Minimum clear space around the logo equals the width/height of the logomark on all sides.

### Colour variations
- **Monochromatic light**: White logomark + white logotype on Lithium or dark backgrounds
- **Monochromatic dark**: Dark logomark + dark logotype on light backgrounds

## 4. Typography

### Brand fonts (from brand guidelines)

| Font | Style | Use |
|------|-------|-----|
| **IBM Plex Sans** | Bold, Regular, Light | Primary sans-serif. Body text, UI, general communications |
| **IBM Plex Mono** | Bold, Regular, Light | Accent monospace. Titles, code snippets, technical content |

### Website fonts (pydantic.dev implementation)

| Font | CSS variable | Weights | Use |
|------|-------------|---------|-----|
| **ABCDiatype** | `--font-diatype` | 400 (Regular), 500 (Medium) | Body text |
| **Dwight** | `--font-dwight` | 500 (Medium) | Headings |
| **FTSystemMono** | `--font-mono` | 500 (Medium) | Monospace / code |
| **ABCFavorit** | `--font-favorit` | 500 (Medium) | Accent / button text |

OpenType features enabled:
- Diatype: `"ss01" on, "ss12" on, "liga" off`
- Dwight: `"ss12" on, "ss10" on, "liga" off`

## 6. Key visual principles

- **Lithium (`#E520E9`)** is the single most distinctive brand colour. It anchors the logomark and primary interactive elements.
- Light mode defaults to **Sugar** (`#FBFFEA`) backgrounds, not pure white. The warm, faintly yellow-green tone is intentional.
- Dark mode defaults to **Dark Purple** (`#36182D`) per brand guidelines, though the website also uses **Petroleum** (`#092224`) for Logfire-specific and product pages.
- **Calcium** (`#FF6550`) provides warmth and energy as the secondary accent, used for highlights, decorative diamonds, and code syntax.
- The palette avoids pure black (`#000000`) and pure white (`#FFFFFF`) in most contexts, preferring the tinted neutrals above.
- Monochromatic logo versions are preferred when the full-colour lockup would clash with background colours.
