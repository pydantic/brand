# Pydantic Logos & Brand Guidelines

## The AI engineering stack

### Pydantic is all you need

- **Pydantic AI** - Agent framework for building production-grade AI applications
- **Pydantic Logfire** - Observability platform for Python applications
- **Pydantic Validation** - Data validation using Python type hints


## Assets

Root folder contains Pydantic brand assets. Product-specific assets are organized in subdirectories:
- `pydantic-ai/` - Pydantic AI logos
- `logfire/` - Pydantic Logfire logos
- `pydantic-validation/` - Pydantic Validation logos
- `icons/` - Product iconography
- `favicon_package/` - Favicon files for web integration
